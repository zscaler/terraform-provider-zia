# zia_http_header_action_profile (Data Source)

* [Official documentation](https://help.zscaler.com/zia/about-http-header-profile)
* [API documentation](https://help.zscaler.com/zia/about-http-header-insertion-profile)

Use the **zia_http_header_action_profile** data source to get information about a HTTP Header insertion Profile. This data source can then be associated with a ZIA and URL filtering rules.
