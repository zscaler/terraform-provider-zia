# zia_http_header_action_profile (Data Source)

* [Official documentation](https://help.zscaler.com/zia/about-http-header-profile)
* [API documentation](https://help.zscaler.com/legacy-apis/http-header-control#/httpHeaderProfile-get)

Use the **zia_http_header_action_profile** data source to get information about a HTTP Header Profile. This data source can then be associated with a ZIA and URL filtering rules as part of the attribute `http_header_profiles`
